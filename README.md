## Instalação

### Clone do repositório

```bash
$ git clone https://github.com/LuisPeixoto/finz-mobile

$ cd finz-mobile
```

### Instalação das dependências

```bash
$ yarn install
```

### Execução

```bash
$ yarn android
```
